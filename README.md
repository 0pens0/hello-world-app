# Hello World App

A simple, beautiful Hello World application built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Interactive greeting in multiple languages
- Modern gradient styling
- Animated interactions

## Deployment

This app is designed to be deployed to Cloud Foundry using the staticfile buildpack.

```bash
cf push
```

## Local Development

Simply open `index.html` in your browser to view the application.
